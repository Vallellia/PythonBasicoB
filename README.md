# practica 2 - conceptos basicos de base de datos
# 1. que es una base de datos relacional?
es un sistema para almacenar informacion organizada en tablas que se pueden relacionar entre si.

# 2. que es una tabla en el contexto base de datos relacionales? 
es como una hoja de calculo formal y con disciplina. organiza datos en columnas (campos) y renglones (registros).

# 3. que es una relacion en el contextos de base de datos relacionales?
es el vinculo entre dos tablas.

# 4. que representa una columna en el contexto de base de datos relacionales?
representa atributo o caracteristica
. nombre
. edad
. matricula

# 5. que representa un renglon en el contexto de base de datos relacionales?
representa un registro individual.

# 6. que es un motor de base de datos relacionales?
es el software que permite crear, administrar y consular la base de datos.

# 7. cuales son los motores de bases de datos relacionales mas populares?
. MySQL
. PostgreSQL
. Microsoft SQL Server
. Oracle Database
. MariaDB

# 8. cual es el motor de base de datos ligero que no necesita un servidor?
vive dentro de una aplicacion, no necesita instalacion de servidor.

# 9. que significa SQL en el contexto de base de datos relacionales?
es el lenguaje que se usa para hablar con la base de datos: 
. SELECT (consultar)
. INSERT (agregar)
. UPDATE (modificar)
. DELETE (borrar)

# 10. cuales son los elementos de un sistema de informacion basado en base de datos relacionales?
. Hardware
. software
. base de datos (tablas, relacionales, datos)
